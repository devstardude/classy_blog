# Classy Blog 

This is a Blog posting app using Node.js, Express, Passport, Mongoose, EJS and some other packages.

### Version: 1.0.0

### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
```

### MongoDB

Open "config/keys.js" and add your MongoDB URI, local or Atlas
